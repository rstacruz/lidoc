Models: Project
¶

Holds indexes of Pages and Files.

This is also the output of parse().

{
  pages: {
    'App': { Page },
    'App: Builder': { Page },
    ...
  },
  files: {
    'hi.html': { File },
    'hello.html': { File },
    ...
  },
  fileTree: { Filetree }
  pageTree: { Pagetree }
}

The keys of pages and files are their id fields.

 
Page = require './page'
File = require './file'

    @set options
