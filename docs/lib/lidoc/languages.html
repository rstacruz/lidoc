getLanguage()
             		                            
             		           
                               
                            
                 ¶               
               
Get the current language we're documenting, based on the extension of the
filename source.



getLanguage(".py")
#=> {name: "Python", commentMatcher: "..", commentFilter: "...", }

             		                            
getLanguage = (source) -> languages[path.extname(source)]
